Detailed Description

The DISOMaster class is a simple wrapper around libisoburn.

DISOMaster provides basic optical drive operation and on-disc filesystem manipulation.

All method calls in this class are synchronous: they do not return until the operation completes. Note the signal is emitted from a separate thread (while the job is actually running).
